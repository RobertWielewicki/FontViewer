# FontViewer
Web application that will allow to dynamically check if the font is valid or compatible with browser.
You can attach this application to your existing website project by simply changing "path" variables in both 'resources/functions.php' and 'resources/scripts.js' to whatever you want remembering, that the both files relate to 'index.php'. So if you have project in directory 'Example' you should rename both variables to 'path = "../Example/fonts"' or wherever you have fonts.
